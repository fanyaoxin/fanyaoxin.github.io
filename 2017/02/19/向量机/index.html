支持向量机 support vector machine

  • 二元分类器
  • 一些特征空间使用线性分割
  • 计算变量 而不是 学习变量

目标是：

构造一个超平面的分割（高维分割），这个分割距离两个class里面最近的元素都尽量远。

distance between hyperplane and support vectors —— margin

即：找到函数变量，使得

  • 在区分训练集上没有错误
  • margin最大化

linear SVMs 线性向量机

对于一般的线性判别分析,

z (distance) = |f(x)|/ ||w||

对于SVM , z=1/ ||w||

所以，为了最大化margin，我们可以 min ||w||, min 1/2 ||w|| **2

————这样就可以求的 合适的w （用拉格朗日啥子方法来求最值） （数学分析里有）

Non-separable Data

去处理非线性分割数据，限制要放松一些

非线性分割数据，主要分成三类：

  • 在SV之外

  • 在SV之中，但是分类对了

  • 在SV之中，但是分类错了

    为了尽量分队，我们需要最小化 yipexilo的和

    这就是 soft-margin SVM

    最有目标： cyipexilo的和+ 1/2 ||w|| *2

非线性 支持向量机 Nonlinear SVMs

  1. 能处理线性分割数据或近似线性分割数据
  2. 也能处理非线性分割数据，也要非线性分割数据，能通过一个映射函数，转化为 线性分割数据

引入kernel function 来满足mercer’s therorem, 使得能够投影高维特征

没有系统的方法来决定最好的kernel function 以及它的参数，通常通过trial 和error 来确认。
