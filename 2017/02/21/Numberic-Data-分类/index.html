Numeric Data 分类

Feb 21, 2017 | Hits

classification

  • 每个instance 都labelled
  • learn how to label or classify unseen examples (finding a rule)
  • two main steps: (主要针对Numeric)
    • training
      • select a portion of the existing data (training data)
      • initialise the wi’s (to random values)
      • apply the rule to the training set and (e.g.) count how many answers are correct ——— score
      • adjust the wi’s (acound to some method)
      • iterate back to step3 until the score is ‘good enough’
    • evaluation
      • select test set
      • apply the rule to the test set and compute the score
  • model
    • correlation and causation 是不同的
    • 两个变量可能相关，但可能不是因果关系 (隐变量)
      • regression, when y is numeric
      • classification, when y is nominal or categorical
      • parametric, if we make assumptions about the distribution
      • non-parametric, if we do not make assumptions about the distribution (later)

  • linear models

    • can be used for classification problems by drawing a decision boundarys

​

目标是找到参数 w向量使得，训练集在线性方程上的误差最小

  • 用线性模型来分类

    • 让回归方程=0 ： 对于instances属于class

      ​ =1 ： 对于instances不属于class

    • 对于要预测的instance带入方程，看最接近0/1

  • 线性模型的结构

    • piecewise linear model
    • non-linear spline model
    • tree structure

  • 逻辑回归

    对于一些数据集，效果比线性回归更好。

    比如说：当最小方差误差不是统计独立的，或者说不是正太分布的

    • 逻辑回归使用对数函数来进行转换，

  • 训练线性回归模型

    • residuals (sum of squares of residuals)

    • two approaches to fit a linear regreesion (adjust the parameters)

      • gradient descent

        using iterative search

        • batch gradient descent (总的)
        • stochastic gradient descent

      • ordinary least squares

        solve the problems analytically

        如果cost函数，是凸函数并且有全局最小值

perceptrons

another type of numeric model

  1. single-layer perceptrons
  2. multi-layer perceptrons (neural network)

  • perceptron learning rule

    is an algorithm for finding a hyperplane

  • A threshold function can ensure that output is either 0 or 1.

  • multi-layer perceptrons

    is essentially a simple kind of neural network, with:

    • input vector

    • intermediate ‘hidden’ values

    • threshold function

    • output value

      神经网络造成的分割是不平整的，意味着可以更加好的模拟现实的情况

    ​

    • each layer is evaluated in sequence

      每一层都是按数列来evaluated的

    • training process

      • follow a steepest descent approach
      • search the landscape of possible output values, and we want to descend from large errors to small (ideally 0) error
      • weights are adjusted by moving backwards:
        • first from the hidden <— output layer
        • and then from the input <— hidden layer
      • the amount of adjustment is proportional to the value of the error function
        • big errors => big adjustments
        • small errors => small adjustments
      • the error rate should decline during the training process

probailistic models

probability theory ——— the interpretation of probability

probability calculus —— computing probabilities

  • two viewpoints

    • a frequentist view: 频率学派

      大量重复试验后，趋向一个分布———— 参数是存在，而且固定的

    • a sujective view： 客观学派，从客观的角度观察

      又叫：贝叶斯学派

  • features in a joint density plot:
    • peak (most frequent value , mode)
    • valley (least frequent value)
    • hole or gap (missing value)
